# Crackme Challenge

## Goal

`bin/crackme_x64.exe` is a multi-stage crackme.
If all stages pass, it prints a flag and waits before exiting.
If any stage fails, it exits immediately with code 1.

Extract the flag.

## Notes

- No PDB symbols. You must work from disassembly only.
- x86 version also available: `bin/crackme_x86.exe`
